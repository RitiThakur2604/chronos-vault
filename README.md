# Chronos Vault ⏳
A full-stack Node.js and Express application that allows users to create time-locked digital time capsules. 

## Features
* **Time-Locked Encryption:** Messages cannot be viewed until the specified unlock date.
* **PIN Security:** Adds a secondary layer of authentication to access capsules.
* **Intrusion Logging:** Tracks and logs unauthorized attempts to open capsules early or with the wrong PIN.
* **Dark Mode UI:** Fully styled EJS frontend.

## Tech Stack
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Frontend:** HTML, CSS, EJS
